## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/Luna-AI.git`
3. Install dependencies: `cd client && npm install`
4. Create a branch: `git checkout -b feature/your-feature`

## Development

```bash
npm run dev      # Start development server
npm run build    # Build for production
npm run lint     # Run linter
```

## Code Style

- Use TypeScript for all new code
- Follow existing component patterns
- Use Tailwind CSS for styling
- Add proper types for all props and state

## Pull Request Process

1. Update documentation if needed
2. Test your changes thoroughly
3. Create a descriptive PR with screenshots if UI changes
4. Wait for review

## Opik Integration

When adding AI features:
- Always trace responses through Opik
- Implement safety guardrails
- Never allow medical prescriptions
- Test with edge cases

## Questions?

Open an issue or reach out to the team!
